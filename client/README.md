# Client App (React + Vite)

This is a modern frontend client built with **React** and **Vite**, styled using **Tailwind CSS**, and supporting real-time features via **Socket.IO**.  
The project is optimized for fast development, clean code, and scalability.

---

## Tech Stack

- **React 19** – UI library for building user interfaces
- **Vite** – Fast build tool and development server
- **Tailwind CSS** – Utility-first CSS framework
- **React Router DOM** – Client-side routing
- **Axios** – HTTP client for API requests
- **Socket.IO Client** – Real-time communication
- **React Hot Toast** – Toast notifications
- **ESLint** – Code linting and quality checks

---

## Dependencies Explained

### Runtime Dependencies

- `react`, `react-dom` – Core React libraries
- `react-router-dom` – Routing and navigation
- `axios` – API calls
- `socket.io-client` – Real-time events (chat, notifications, etc.)
- `react-hot-toast` – User-friendly toast notifications
- `tailwindcss` – Styling framework
- `@tailwindcss/vite` – Tailwind integration with Vite

### Development Dependencies

- `vite` – Development server & bundler
- `@vitejs/plugin-react` – React support for Vite
- `eslint` + plugins – Linting and best practices
- `@types/react`, `@types/react-dom` – Type definitions for React

---

## Installation

Available scripts in `package.json`:

```bash
git clone https://github.com/PhucIT226/Chat-app
cd client
npm install
npm run dev
cd ../server
npm run server
```
